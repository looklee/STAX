📊 数据收集说明

STAX 应用数据收集政策

1. 概述

欢迎使用STAX应用！本数据收集说明详细说明了我们如何收集、使用和保护您的个人信息。 我们致力于透明地向您披露我们的数据处理方式，并确保您对自己的信息有充分的控制权。

2. 我们收集哪些数据

2.1 设备信息

为了提供最佳的应用体验，我们会收集以下设备信息：

  • 设备型号和操作系统版本 - 用于优化应用性能
  • 设备唯一标识符 - 用于追踪应用崩溃和错误
  • 处理器和内存信息 - 用于性能监控
  • 显示屏分辨率 - 用于适配界面显示

2.2 应用使用数据

我们收集应用的使用统计信息，以改进用户体验：

  • 应用启动和关闭时间 - 用于了解用户行为
  • 功能使用情况 - 追踪哪些功能被最常使用
  • 崩溃报告 - 当应用异常崩溃时
  • 性能指标 - 帧率、延迟等性能数据

2.3 位置信息

⚠️ 位置权限说明：

应用可能请求访问位置信息权限。此权限仅在用户明确授予时使用，用途限于：

  • 增强AR/MR场景的地理感知效果
  • 优化VR体验中的空间定位

我们不会将位置数据用于商业目的或与第三方共享。

2.4 摄像头和音频权限

应用可能请求以下权限：

  • 摄像头权限 - 仅用于MR（混合现实）模式，让您能看到真实环境
  • 音频权限 - 用于音效反馈和声音提示

这些权限仅在用户明确授予时才会被请求和使用。

2.5 用户内容

应用本地存储用户在VR环境中创建的内容（如生成的几何体、颜色选择等）。 这些数据仅存储在您的设备上，不会上传到服务器。

3. 数据的使用目的

收集的数据 使用目的 法律依据
设备信息 优化应用性能，确保兼容性 应用正常运行的必要
应用使用数据 改进功能，分析用户行为 合法利益
崩溃报告 诊断和修复问题 应用稳定性的必要
位置信息 增强MR/VR体验 用户明确同意
摄像头/音频 MR模式、音效反馈 用户明确同意

4. 数据的收集方式

我们通过以下方式收集数据：

  • 直接收集 - 应用运行时自动收集的设备和使用信息
  • 用户授权 - 仅在用户明确授予权限时收集位置、摄像头和音频数据
  • 分析工具 - 使用匿名分析服务跟踪应用性能
  • 错误报告 - 应用崩溃时收集诊断信息

5. 数据的存储和安全

5.1 存储位置

  • 大部分数据存储在您的PICO设备上
  • 性能指标和崩溃报告可能通过加密连接传输到我们的安全服务器
  • 所有传输都使用HTTPS加密

5.2 数据安全措施

  • 采用行业标准的加密技术保护数据
  • 实施访问控制，限制只有授权人员能访问数据
  • 定期进行安全审计和评估
  • 遵守数据保护法规

6. 数据保留

我们会根据以下原则保留数据：

  • 本地数据 - 除非您手动删除，否则保存在设备上
  • 分析数据 - 通常保留12个月后自动删除
  • 崩溃报告 - 保留30天用于分析和改进
  • 性能指标 - 保留90天用于监控应用健康状况

7. 数据共享

✅ 重要保证：

我们不会与第三方共享您的个人数据用于营销或商业目的。 数据仅在以下情况下被共享：

  • 遵守法律或政府要求
  • 保护我们的权利和安全
  • 防止欺诈或非法活动
  • 获得您的明确同意

8. 儿童隐私保护

STAX应用不特意收集13岁以下儿童的个人数据。 如果我们意识到收集了儿童的数据，我们将立即删除这些信息。

9. 用户权利

根据适用的数据保护法，您有权：

  • 访问权 - 了解我们收集的关于您的数据
  • 更正权 - 更正任何不准确的数据
  • 删除权 - 请求删除您的数据
  • 限制权 - 限制我们对数据的使用
  • 撤销权 - 撤销之前的同意
  • 数据可携权 - 获取您的数据副本

要行使这些权利，请联系我们的隐私团队。

10. 国际数据传输

如果您位于欧盟或其他地区，您的数据可能被传输到其他国家/地区进行处理。 我们承诺采取措施确保您的数据继续受到相同水平的保护。

11. 第三方服务

应用可能使用第三方服务来帮助我们提供和改进服务，这些服务包括：

  • 分析服务 - 用于追踪应用使用情况
  • 崩溃报告工具 - 用于诊断技术问题
  • 云服务 - 用于性能优化（可选）

这些第三方服务提供商同样受到严格的数据保护协议约束。

12. 政策更新

我们可能不时更新本数据收集说明以反映法律变化或我们实践的改进。 我们会在进行重大更改时通知您。

13. 联系我们

如果您对本数据收集说明有任何疑问或关切，请通过以下方式与我们联系：

  • 电子邮件 - GitHub Issues
  • 应用内反馈 - 通过应用的帮助菜单
  • 隐私请求 - GitHub Issues
