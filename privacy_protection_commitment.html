STAX 隐私和数据保护承诺书

我们的核心承诺

STAX致力于保护您的隐私和个人数据。我们相信透明度和信任是建立良好用户关系的基础。 本承诺书详细阐述了我们如何尊重和保护您的隐私权。

1. 隐私优先原则

隐私不是事后考虑，而是我们设计和开发STAX应用时的首要考虑。 我们采用"隐私设计"方法，确保从项目开始阶段就纳入隐私保护。

2. 我们的核心隐私原则

🎯

最小数据原则

我们仅收集提供应用功能所必需的最少数据量。 不收集不必要的信息，即使这些信息可能对我们有商业利益。

🔒

用户控制

您对自己的数据拥有完全控制权。我们尽一切努力让您能够轻松访问、 修改或删除您的个人信息。

🔐

安全保护

我们实施业界最佳的安全措施来保护您的数据免受未经授权的访问、 修改或泄露。

🌍

透明沟通

我们以清晰、简洁的语言解释我们的隐私实践。 没有隐藏的条款或模糊的措辞。

3. 具体承诺

3.1 数据收集承诺

  • 仅收集必需数据 - 我们不会收集超出提供应用功能所需的任何数据
  • 明确告知用户 - 在收集任何个人数据前，我们会清楚地告知您
  • 获取明确同意 - 对于敏感数据（如位置、摄像头、音频）， 我们会获得您的明确同意
  • 合法和正当目的 - 所有数据收集都只用于合法和正当的目的

3.2 数据使用承诺

  • 不用于营销 - 我们不会将您的数据用于不相关的营销目的
  • 不出售数据 - 我们永远不会出售您的个人数据给第三方
  • 仅与必要方共享 - 如果需要与第三方共享数据， 我们会获得您的明确同意或仅与严格需要的服务提供商共享
  • 明确的目的限制 - 数据仅用于告知您的目的，不扩展到其他用途

3.3 数据安全承诺

  • 加密传输 - 所有敏感数据在传输时使用SSL/TLS加密
  • 安全存储 - 数据在服务器上使用业界标准加密保存
  • 访问控制 - 仅授权员工可以访问您的数据，且受严格限制
  • 定期审计 - 我们定期进行安全审计和评估
  • 事件响应 - 如果发生数据泄露，我们会按法律要求及时通知您

3.4 用户权利承诺

  • 访问权 - 您有权了解我们持有的关于您的所有数据
  • 更正权 - 您可以要求我们更正任何不准确的信息
  • 删除权 - 您可以要求删除您的数据（除非法律另有规定）
  • 撤销权 - 您可以随时撤销之前给予的任何同意
  • 投诉权 - 您有权向相关数据保护机构投诉

3.5 儿童保护承诺

  • STAX不会故意收集13岁以下儿童的个人数据
  • 如果我们意识到收集了儿童数据，我们会立即采取措施删除这些信息
  • 我们建议父母和监护人监督儿童使用本应用

4. 技术措施

我们采取的安全技术措施：

  • 端到端加密 - 某些数据在设备级别加密，即使我们也无法读取
  • 安全协议 - 使用最新的安全通信协议
  • 防火墙和入侵检测 - 多层网络安全保护
  • 定期更新 - 保持系统和软件最新，修补已知漏洞
  • 备份和恢复 - 确保数据可以从灾难中恢复

5. 法规遵守

STAX承诺遵守所有适用的数据保护法律和法规，包括但不限于：

  • 中国网络安全法和个人信息保护法 (PIPL)
  • 欧盟通用数据保护条例 (GDPR)
  • 加州消费者隐私法 (CCPA)
  • 其他各地区适用的隐私和数据保护法

6. 员工培训和责任

  • 所有可以访问用户数据的员工都必须接受隐私和数据保护培训
  • 员工必须签署保密协议和数据保护承诺
  • 违反隐私政策的员工将面临纪律处分

7. 数据保留和删除

✅ 我们的保留政策：

  • 数据仅保留其所必需的时间
  • 删除前您可以请求下载您的数据
  • 对于不活跃账户，我们会在12个月后进行安全删除
  • 法律要求的保留例外情况会清楚地告知您

8. 第三方服务提供商

当我们与第三方共享您的数据时，我们确保：

  • 第三方与我们签署数据处理协议，承诺相同水平的隐私保护
  • 第三方不得将数据用于自己的目的
  • 我们对第三方的数据安全实践负责

9. 国际数据传输

如果您的数据被传输到其他国家/地区，我们确保：

  • 采取必要的法律和技术措施保护您的数据
  • 遵守数据传输的国际法律要求
  • 在必要时获取您的明确同意

10. 隐私政策更新

  • 本承诺书和隐私政策可能会不时更新
  • 我们会提前通知您任何重大变化
  • 您有权拒绝新政策并停止使用应用

11. 您如何保护自己

  • 使用强密码 - 如果应用有账户功能，使用强密码
  • 审查权限 - 定期审查应用在您的设备上被授予了哪些权限
  • 更新应用 - 及时更新应用以获取最新的安全修复
  • 报告问题 - 如果您发现任何隐私问题，请立即报告

12. 联系隐私团队

如果您对我们的隐私实践有任何疑问或关切，请通过以下方式与我们联系：

  • 隐私电子邮件 - GitHub Issues
  • 支持电子邮件 - GitHub Issues
  • 应用内反馈 - 通过应用的帮助菜单
  • 邮寄地址 - 请在隐私电子邮件中申请

13. 隐私保护保证

✓ 我们对隐私保护的承诺是认真和永久的。隐私不仅仅是一项政策， 而是我们核心价值观的一部分。

我们欢迎您的监督和反馈。如果我们未能达到这些承诺， 请让我们知道，我们将立即采取行动纠正。
